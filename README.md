# Sales ETL & Business Intelligence Project

## Project Overview

This project demonstrates a complete end-to-end Business Intelligence solution using Microsoft BI Stack tools.

The project covers the entire BI workflow, starting from extracting raw data from Excel files, transforming and loading the data into SQL Server using SSIS, building a Data Warehouse based on Star Schema design, and finally creating interactive dashboards in Power BI to generate meaningful business insights.

The main objective of this project is to analyze sales performance, customer behavior, product profitability, geographic trends, and time-based performance metrics.

---

## Tools & Technologies

- Microsoft SQL Server
- SQL Server Integration Services (SSIS)
- Power BI
- DAX
- Microsoft Excel

---

## Project Workflow

1. Extracted raw sales data from Excel files.
2. Built ETL pipelines using SSIS to load and transform data.
3. Performed data cleaning and transformation in SQL Server.
4. Designed a Data Warehouse using Star Schema architecture.
5. Created Fact and Dimension tables.
6. Built data models and relationships in Power BI.
7. Developed KPIs and DAX measures for business analysis.
8. Designed interactive multi-page dashboards.
